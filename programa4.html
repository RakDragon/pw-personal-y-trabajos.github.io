Programa #4: Búsqueda en Vector

Elaborar un programa en JAVA que realice una búsqueda de un número en un vector, mostrando solamente un mensaje indicando cuántas veces apareció el número, además de imprimir el vector.

  • Ejemplo:
Valor a buscar: 4
"El número 4 aparece 2 veces en el vector:"
(1 4 7 9 21 4 0 2 5 2)

Código:

import java.util.Scanner;

public class tarea19 {
 
  public static void main(String[] args) {
    try (Scanner s = new Scanner(System.in)) {
        int[] n = new int[10];
        int buscar;

        System.out.print("Para el siguiente vector, debe de introducir 10 dígitos
        (se recomienda que digite por lo menos un número y que este se repita 2 o
        más veces).\n");
        // Leemos el array
        for(int i=0;i<10;i++) {
          System.out.print("- Introduzca el número " + (i+1) + ": ");
          n[i] = Integer.parseInt(s.nextLine());
        }
 
        // Leemos el número a buscar
        System.out.print("Introduzca el número a buscar: ");
        buscar = Integer.parseInt(s.nextLine());
 
        // Buscamos el número en el array
        int cuenta = 0;
        for(int i=0;i<10;i++) {
          if(n[i] == buscar) {
            cuenta++;
          }
        }
 
        // Mostramos el resultado de la búsqueda
        System.out.print("- El vector es: ");
        for(int i=0;i<10;i++) {
          System.out.print(n[i] + " ");
        }
        System.out.println();
        if(cuenta == 0) {
          System.out.println("- El número no se encuentra en el vector");
        } else {
          System.out.println("- El número " + buscar + " aparece " + cuenta + " veces
          en el vector.");
        }
    } catch (NumberFormatException e) {
        e.printStackTrace();
    }
  }
}
